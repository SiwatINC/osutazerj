# osu!tazer
This repository house the resources for building osu!tazer.
BOMs:
  - Arduino Yun Rev 2.0
  - Arduino Motor Shield Rev 3.0
  - Geared DC Motor
  - A 3D printer with ~ 243g of Fillament
  - Generic Discharge-based High Voltage Generator
